# Checklist diário

- [ ] acessar a UI de pelo menos um nó do cluster;
- [ ] validar `pvecm status`;
- [ ] conferir se todos os nós estão online;
- [ ] verificar `pvesm status`;
- [ ] observar se há erro recente em `pveproxy`, `pve-cluster` ou `corosync`;
- [ ] registrar qualquer anomalia percebida.
