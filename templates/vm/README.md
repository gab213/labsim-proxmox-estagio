# Templates de VM

Este diretório documenta o padrão de templates usados no cluster do LabSim.

## Perfil inicial recomendado

- nome: `tmpl-ubuntu-redes-moveis`
- 4 vCPU
- 16 GB de RAM
- 100 GB de disco
- bridge `vmbr0`
- Ubuntu Server LTS ou Debian

## Uso esperado

- criação rápida de ambientes de aula;
- padronização de laboratório;
- redução de improviso em VMs novas;
- suporte a stacks abertas de redes móveis.
